Projets

E-Ticket

Semitan

J'ai développé des web-services en Php Symfony pour l'application mobile de la SEMITAN. Lorsqu’un client fait une demande de duplicata de sa carte (vol, perte), le temps que l'on lui délivre une nouvelle carte, un code lui est envoyé. Une fois ce code saisi sur l'application un ticket est créé. Le ticket est valable jusqu'à ce que le client reçoit sa nouvelle carte. Les web-services permettent le contrôle de ce code(format, existant... )ainsi que le compostage du ticket.

Juillet 2018 - Septembre 2018
